## Usage

### Customizing Content

Modify the `SERVICES` array at the top of the file to add your own sections:

```tsx
// Change Here
const SERVICES = [
  {
    id: "01",
    title: "Web Design",
    description: "Creating beautiful digital experiences.",
    image: "https://images.unsplash.com...",
  },
  // ...
];
```

```tsx
import VerticalTabs from "@/components/vertical-tabs";

export default function Page() {
  return <VerticalTabs />;
}
```

## Features

- **Progress Indicators**: Visual progress bars show auto-play status on active items.
- **Content Synchronization**: Updates image and text content based on the selected item.
- **Auto-cycle**: Automatically rotates through the collection (pauses on hover).
- **Smooth Transitions**: Slide animations for content changes.
- **Interactive Navigation**: Click to jump to any item in the collection.
